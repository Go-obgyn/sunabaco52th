# AGENTS Instructions for SUNABA CAFE

- This repository is for **SUNABA CAFE**'s single-page static website.
- Use only **HTML / CSS / Vanilla JavaScript** (no frameworks, no build tooling).
- Follow a **mobile-first** approach.
- Avoid flashy effects; keep interactions quiet and minimal.
- Keep the design **photo-first** with calm, refined spacing.
- Hero must use lightweight scrollytelling only: **sticky container + opacity transitions**.
- Do **not** implement a hamburger menu in v1.
- Do **not** use Google Maps JavaScript API; use a plain external link button.
- Images must keep sharp corners (**no rounded corners**).
- Ensure no horizontal scroll at **375px** viewport width.
- Respect `prefers-reduced-motion` and reduce/disable motion accordingly.
- Layout must still work with placeholder images when real images are missing.
- Phone number is not finalized, so do not add a clickable `tel:` link.
- Instagram QR can remain a temporary placeholder image.
